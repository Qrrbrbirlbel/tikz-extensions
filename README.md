## Introduction

This is not a LaTeX package but a collection of libraries for PGF and TikZ.

These are

 * `transformations.mirror`,
 * `paths.arcto`,
 * `paths.ortho`,
 * `paths.timer`,
 * `patterns.images`,
 * `topaths.arcthrough` and
 * `misc`.
